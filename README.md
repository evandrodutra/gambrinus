# node-skeleton :skull:
========

**node-skeleton** was made to have a simple skeleton for a complete Web App ready to go

## Installation

- Perform a clone of this repo.
- Install packages (`npm install`)

## Technologies

* babel
* eslint
* lodash
* moment
* webpack
* webpack-dev-server
* loaders:
  * css
  * file
  * html
  * sass
  * postcss
  * raw
  * style
  * url

## Scripts

* `npm run build`
* `npm run lint`
* `npm run test`
* `npm run clean`
* `npm run start`
